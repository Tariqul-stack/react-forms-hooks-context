# React Forms, Hooks & Context

This repository contains my learning and practice of core React concepts including form handling, custom hooks, and Context API.

## 📚 Topics Covered

* Controlled and Uncontrolled Components
* Handling form data using `formData`
* Form validation and error handling
* Creating and managing product forms
* Displaying form data in tables
* Custom Hooks to reduce code duplication
* Component tree structure in React
* Introduction to Context API
* Creating and using simple context
* Exploring advanced Context API usage

## 🛠️ Tech Stack

* React
* JavaScript (ES6+)
* CSS / Tailwind (if used)

## 🎯 Purpose

The goal of this repository is to strengthen my understanding of core React concepts by building small examples and mini projects.
This is part of my journey to become a Full Stack Developer and prepare for remote developer opportunities.
